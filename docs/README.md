### 关于我们

#### 前端kendall介绍
Kendall是一名在校生，目前在国内某高校读研，开源本项目是自己个人想通过开源平台帮助更多想学习前端的小伙伴，同时也有助于自己系统性的学习和总结，当然也希望能结交到同行跟多优秀的大咖。
